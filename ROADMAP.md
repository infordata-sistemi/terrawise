# TerraWise roadmap

> Public-facing roadmap. The internal coordination / sprint planning lives elsewhere; this file is what we're comfortable saying out loud about direction.

## Now

- Hardening the Karst Firewall 5.0 pilot deployment ahead of the 14 August 2026 Interreg close.
- Rolling out the IoT e-nose sensor network across the two pilot municipalities (Duino Aurisina — Devin Nabrežina + Miren-Kostanjevica).
- Tightening the integrated-surveillance loop (e-nose · thermal IR · drones · satellite · citizen reporting → one dispatch).
- Preparing PyroWISE for a first open scientific publication.

## Next

- Second TerraWise deployment site — onboarding a non-Karst territory to validate the calibration pipeline.
- Public REST API for the KFWI service (rate-limited, citation-required).
- Cross-border alert federation (programme-level adoption beyond the IT–SI Karst).

## Later

- Multi-hazard extensions beyond wildfire (the *T* in TerraWise — territorial emergencies more broadly).
- Open benchmark suite for fire-spread simulators (PyroWISE evaluation against community datasets).
- Industry-5.0 / citizen-science feedback loops contributing back to the KFWI training set.
